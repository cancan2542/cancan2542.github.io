# ハブサイト実装依頼プロンプト

あなたはフロントエンドエンジニアです。
以下の要件・制約を満たす **リンク集（ハブサイト）** を実装してください。

## 目的

* 個人の発信活動（車中泊・ブログ・YouTube）へのリンク集サイトを作る
* コンテンツ消費は行わず、送客に特化する

## 技術要件

* ビルド不要
* 純粋な **HTML / CSS / JavaScript（最小限）**
* GitHub Pages でそのまま公開可能
* 外部ライブラリは使用しない（CDN含む）
* フレームワーク（React / Vue 等）は使用しない

## デザイン要件

* lit.link に近い **カード型UI**
* モダン・ミニマル
* SPファースト（スマホ最優先）
* ダークテーマ前提
* 余白・タイポグラフィを重視
* フォントはシステムフォントでOK

## 構成

* 1ページ完結
* 以下ファイルを出力すること

  * index.html
  * style.css
  * main.js（不要であれば省略可）

## HTML構造方針

* semantic HTML を使用（header / main / section / footer）
* 将来 React 化しても流用できる DOM 構造

## コンテンツ

* 表示するリンク・文言は `content.md` を参照すること

## その他

* コードは可読性を重視
* クラス名は意味ベースで命名すること
