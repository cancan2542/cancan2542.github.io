# SNSプロフィール用リンク集ページ設計プロンプト

## 目的

SNSプロフィール（X / Instagram / YouTube 等）から遷移する
**個人用リンク集ランディングページ**を作成する。

使用技術は **HTML / CSS / JavaScript のみ** とし、
フレームワークやビルドツールは使用しない。

---

## コンセプト

* 白 × グレー基調
* モノトーンでミニマル
* 余白を活かした洗練されたデザイン
* 情報量は少なく、直感的に操作できる

---

## 必須要件

* モバイルファースト
* 完全レスポンシブ対応
* 1ページ完結（縦スクロール）
* JavaScriptは最小限（なくても成立する構成）

---

## ページ構成（ワイヤーフレーム）

### 1. ヒーローエリア（ページ上部）

### 中央寄せ配置

* アバター画像（円形 or 角丸）
* 表示名（やや大きめ）
* 簡単な説明文（小さめ・グレー）

```text
[ avatar ]
NAME
short description
```

---

### 2. リンクリスト（メインコンテンツ）

* 縦並び
* ボタン型リンク
* タップしやすい高さ

各リンク要素：

* サービス名（主）
* 補足テキスト（任意・小文字）

```text
[ YouTube   | 車中泊・Vlog ]
[ Blog      | ガジェットレビュー ]
[ X (Twitter) ]
[ GitHub ]
```

---

### 3. フッター

* 最小限の情報
* 小さな文字サイズ
* 薄いグレー表示

```text
© 2026 Your Name
```

---

## デザイン仕様

### カラー

* 背景：`#FFFFFF`
* メインテキスト：`#333333`
* サブテキスト：`#777777`
* 枠線：`#DDDDDD`
* ホバー背景：`#F2F2F2`

※ 純黒（`#000000`）は使用しない

---

### フォント

* サンセリフ体
* システムフォント or Webフォント1種まで
* フォントサイズと太さで情報の強弱を表現

---

### ボタン（リンク）

* 横幅：100%
* 枠線あり
* 角丸（軽め）
* hover / tap 時に背景色が変化
* transition：0.15〜0.25s 程度

---

## レスポンシブ設計

### モバイル（最優先）

* 横幅：100%
* 十分なパディング
* 指で押しやすいUIサイズ

### PC / タブレット

* 最大幅を制限（例：480〜640px）
* 中央寄せ
* 余白を増やし、間延びさせない

---

## JavaScript要件

* 必須ではない
* 使用する場合：

  * hover / tap の視覚効果
  * 軽微なアニメーション
* 外部ライブラリは使用しない

---

## 出力要件

* `index.html`
* `style.css`
* `script.js`（必要な場合のみ）

すぐに公開可能な状態で出力すること。
